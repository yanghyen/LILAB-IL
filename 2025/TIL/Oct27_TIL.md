### 객관적 평가 지표(Intern Project)
### accuracy/F1
#### accuracy
- 모델이 전체 데이터 중 정확하게 예측한 비율
- 데이터 불균형이 심하면 모델의 실제 성능 왜곡 가능
- Accuraty = 정확하게예측한샘플수 / 전체샘플수
#### F1 Score
- 정밀도(precision)와 재현율(Recall)의 조화 평균
- 정밀도(precision): 모델이 맞다고 한 것 중 실제 정답의 비율
- 재현율(Recall): 실제 정답 중 모델이 맞다고 예측한 비율
- F1점수는 이 둘 간의 균형 측정 -> 데이터 불균형이 있을 때 정확도보다 더 신뢰
- F1 = 2 * Precision * Recall / Precision + Recall 

### BLEU for MT
- Bilingual Evaluation Understudy: 모델이 생성한 번역문이 사라밍 작성한 하나 이상의 참조 번역문과 얼마나 유사한지 측정
- 모델의 번역문과 참조문 사이의 겹치는 n-그램의 개수를 세어 점수 매김
- 장점: 객관적이고 자동화되어 있으며, 상관관계가 높다는 것 입증됨
- 단점: 문법의 정확성이나 의미의 정확성보다 단어의 일치에 중점을 두어, 자연스러움이나 미묘한 의미 차이를 완벽하게 포착하지 못함
- 점수: 0과 1 사이의 값을 가지며, 1에 가까울수록 번역 품질이 우수함

###  perplexity for LMs
- 혼란도(perplexity)는 모델이 주어진 텍스트(데이터 셋)를 얼마나 잘 예측하는지, 얼마나 확신을 가지고 예측하는지 측정
- 언어 모델이 다음에 올 단어를 예측할 때 겪는 평균적인 어려움 나타냄
    - 수학적 정의: 언어 모델이 테스트 데이터에 할당한 확률의 역수에 대한 기하 평균
    - 값의 해석: 낮을수록 모델이 텍스트의 구조와 패턴을 더 잘 이해하고 다음 단어를 더 정확하게 예측한다는 것
    - 용도: 훈련 과정 중 또는 훈련 후 모델의 일반화 능력 평가

### intrinsic similarity/analogy for embeddings
#### intrinsic similarity
- 내재적 유사성(intrinsic similarity): 모델이 학습한 임베딩 공간에서 두 단어/문장의 유사성이 사람의 판단과 얼마나 일치하는지 측정
    - king과 queen의 임베딩 벡터 간의 거리가 king과 car의 거리보다 가까워야 함
    - 평가 방법: 사람이 미리 점수를 매겨놓은 단어 쌍 유사성 데이터셋(WordSim-353)을 사용하여 모델이 계산한 임베딩 벡터 간의 코사인 유사도와 비교하여 상관관계 측정
- 유추(Analogy): 모델이 임베딩 공간에서 단어 간의 관계를 얼마나 잘 포착하는지 측정
    - 단어 벡터 연산을 이용한 유추 문제
    - A:B = C:?와 같은 유추 문제를 풀게 함. 임베딩 공간에서 B - A + C = ?을 계산하여 결과 벡터에 가장 가까운 단어 ?를 찾는 방식으로 모델의 정확도 평가
